**Commands for AU-tan**

```
.anime - Searches AniList for an anime
.announce - Makes an announcement for everyone
.announce.edit - Edits an existing announcement
.color - Give yourself a coloured username, or clear ther colour on your username
.help - Shows this help
.manga - Searches Anilist for a manga
.me - Display your profile card
.ping - A basic ping command
.reactions.get - Get a list of everyone who has reacted to a message with a specific emoji
.reactions.has_reacted - Check if a user has reacted to a message with a specific emoji
```
